# SA-conf
Use SA-conf in order to quantify the structural variability of a multiple target conformations (MTC) set and localize the structural variable positions and regions of the target.


### Installation
The executables are findable in the github releases. If you want to install the source files, a conda environment is provided for both the latest version of SA-conf on the main branch and the Python 2.7 version. To install a conda environment, use ```conda env create -f env.yaml```
